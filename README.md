# template-interview

A comprehensive interview template project with frontend and backend components.

## Getting Started

### Prerequisites
- Review the README files in both `backend/` and `frontend/` directories for specific setup instructions
- Each project contains detailed guides on environment configuration, required variables, and startup commands

### Setup Instructions

1. **Backend Setup**
    - Navigate to the `backend/` directory
    - Follow the README for dependencies, environment variables, and database configuration
    - Run the application using provided startup commands

2. **Frontend Setup**
    - Navigate to the `frontend/` directory
    - Follow the README for dependencies, environment variables, and build configuration
    - Run the development or production server as specified

### Troubleshooting

If you encounter any issues:
- Check the **Troubleshooting** section in each project's README
- Verify that all environment variables are correctly configured
- Ensure all dependencies are properly installed
- Review project-specific documentation for common issues
